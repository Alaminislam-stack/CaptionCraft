# CaptionCraft Client

An AI-powered social media caption generator built with Vite, React, and shadcn/ui.

## Getting Started

1.  **Install dependencies:**
    ```sh
    npm install
    ```

2.  **Start the development server:**
    ```sh
    npm run dev
    ```

3.  **Build for production:**
    ```sh
    npm run build
    ```

## Tech Stack

-   **Framework:** Vite + React
-   **Styling:** Tailwind CSS + shadcn/ui
-   **State Management:** React Query / Context API
-   **Icons:** Lucide React
